For this reflection I was asked to create a result from a sub query within a query. The image bellow shows the schema of the database for the teams, venues, players, referees, bookings, players, substitutions etc for the Euro 2016.

I decided to find all the players that missed penalties and order them by age.

SELECT player_id, player_name AS Full_Name, age, posi_to_play AS Position
FROM player_mast
WHERE player_mast.player_id IN (
SELECT player_id
FROM penalty_shootout
WHERE score_goal = 'N'
)
ORDER BY age

Here is the breakdown of what I have done

First I have selected the player_id, player_name and age from the player mast table giving a couple of the column names aliases to make them more people freindly:

SELECT player_id, player_name AS Full_Name, age, posi_to_play AS Position
FROM player_mast

Then next I creted a sub/inner query where I linked the player_id to the table penalty_shootout. 

WHERE player_mast.player_id IN (
SELECT player_id
FROM penalty_shootout

Then from here I wanted to find out what what players missed their penalties, I did this by simply selecting 'N' for no in the score_goal column: 

WHERE score_goal = 'N'
)

Next I ordered the players by acending age from youngest to oldest.

ORDER BY age
